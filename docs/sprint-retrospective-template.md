## Agenda (15–20 min)

| Time   | Activity |
|--------|----------|
| 0–2 min | **Set the stage** — Remind everyone: no blame, focus on process and outcomes. |
| 2–6 min | **Went well** — Each person shares 1–2 things that went well this sprint. |
| 6–10 min | **To improve** — What didn’t go well or was frustrating? Be specific. |
| 10–14 min | **Kudos** — Shout-outs to teammates (celebrate wins). |
| 14–18 min | **Action items** — 2–4 concrete things to Start / Stop / Continue next sprint. |
| 18–20 min | **Wrap-up** — Confirm who will post this to Confluence and submit. |
